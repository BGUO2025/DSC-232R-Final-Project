OVERVIEW

Predicting Review Ratings at Scale

This project analyzes Amazon U.S. customer reviews to predict review scores (star_rating) using PySpark for scalable data processing and machine learning. Due to the scale of the dataset, distributed computing techniques were required to efficiently process and analyze the data. The objective is to predict review ratings from textual and metadata features, enabling practical applications such as automated sentiment analysis and product quality monitoring. However, building predictive models at this scale introduces several challenges including class imbalance in ratings, high-dimensional text features, and the computational complexity of large-scale processing.

Challenges

Working with over 102 million reviews introduces challenges such as high-dimensional text data, class imbalance across ratings, and computational overhead for distributed processing. The star rating distribution helps identify these issues and informs feature engineering and model selection.

Conclusion
