OVERVIEW

Predicting Review Ratings at Scale

This project analyzes Amazon U.S. customer reviews to predict review scores (star_rating) using PySpark for scalable data processing and machine learning. Due to the scale of the dataset, distributed computing techniques were required to efficiently process and analyze the data. The objective is to predict review ratings from textual and metadata features, enabling practical applications such as automated sentiment analysis and product quality monitoring. However, building predictive models at this scale introduces several challenges including class imbalance in ratings, high-dimensional text features, and the computational complexity of large-scale processing.

Analysis

Model Fit & Performance

Logistic Regression achieved 60.84% training accuracy and 60.69% validation accuracy. Random Forest improved slightly with 60.98% train and 60.83% validation accuracy, showing minimal underfitting. Feature interactions are better captured with tree-based models.

Random Forest Experiments

We experimented with different hyperparameters: more trees, deeper max depth, and varying feature sampling strategies. The tuned model slightly improved validation accuracy, balancing bias and variance.

Next Steps: Gradient Boosted Trees

Gradient Boosted Trees (GBT) will capture non-linear feature interactions more effectively. Comparing RF and GBT helps understand bias vs variance trade-offs. GBT requires careful tuning and is computationally intensive.

Conclusion

Conclusion

Random Forest models achieved the best performance, with the highest validation accuracy of 0.6103. Feature importance analysis revealed that review length, word count, and helpfulness ratio were the strongest predictors of star ratings.
